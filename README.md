* Ruby version: 3.2.2

#### Installation steps
* install ruby version using rbenv: 3.2.2

`rbenv install 3.2.2`

`rbenv global 3.2.2`

* After cloning the repository run:

`bundle install`

* Setup local db password in database.yml and then run:

`rails db:create`

* run the migration

`rails db:migrate`

* run the server

`rails s`
